=================================================

The HTTP Bridge allows external devices to publish information on the harmonized federated 
network. It is a simple HTTP server that accepts POST requests with JSON payloads. The 
payloads are then published on the network as messages.

More information about the HTTP Bridge can be found in the following resources:

* [Sensor Data API reference](/ReportSensorData) on this gateway.
* [HTTP Bridge repository](https://github.com/Neuro-Foundation/IoTBridgeHttp) on GitHub.
